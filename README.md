# address-book

It's an address book! It lets you add and manage your contacts.

## Technologies Used

- [pnpm](https://pnpm.io)
- [`create-t3-app`](https://create.t3.gg)
- TypeScript
- Next.js
- tRPC
- Tailwind
- Drizzle ORM
- Postgres
- NextAuth.js

## Prerequisites

- [pnpm](https://pnpm.io)

## Configure

- `cp .env.example .env` and update values

## Install

- `pnpm i`

## Develop

- `pnpm dev`
