# address-book

It's an address book! It lets you add and manage your contacts.

## Technologies Used

- Bun
- [`create-t3-app`](https://create.t3.gg)
- TypeScript
- Next.js
- tRPC
- Tailwind
- Drizzle ORM
- Postgres
- NextAuth.js

## Prerequisites

- [bun](https://bun.sh)

## Configure

- `cp .env.example .env` and update values

## Install

- `bun i`

## Develop

- `bun dev`
